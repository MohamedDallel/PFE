Git est un logiciel permettant de suivre les modifications dans n'importe quel ensemble de fichiers , 
généralement utilisé pour coordonner le travail entre les programmeurs développant en collaboration le code source pendant le développement du logiciel . 
Ses objectifs incluent la vitesse, l'intégrité des données et la prise en charge des flux de travail distribués et non linéaires
